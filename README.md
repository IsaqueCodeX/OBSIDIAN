## Visão Geral

O **Obsidian Auto Studio** é uma landing page de alta conversão desenvolvida para um estúdio de estética automotiva premium em São Paulo. O projeto foca em transmitir sofisticação e exclusividade, atendendo a um público exigente que busca serviços de Vitrificação, PPF (Paint Protection Film) e Polimento Técnico.

O objetivo de negócio é a captação de leads qualificados através de uma interface imersiva, performática e otimizada para SEO local.

## Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias de ponta:

-   **React 18**: Biblioteca JavaScript para construção de interfaces.
-   **TypeScript**: Superset do JavaScript que adiciona tipagem estática.
-   **Vite**: Ferramenta de build de nova geração, extremamente rápida.
-   **Tailwind CSS**: Framework CSS utilitário para estilização ágil e responsiva.
-   **Framer Motion**: Biblioteca para animações fluidas e complexas.
-   **Shadcn/UI**: Coleção de componentes reutilizáveis construídos com Radix UI e Tailwind.
-   **Lucide React**: Biblioteca de ícones leve e consistente.

## Funcionalidades Chave

-   **Arquitetura Mobile-First**: Layout totalmente responsivo, garantindo experiência perfeita em qualquer dispositivo.
-   **Design Premium**: Interface sofisticada com uso de "dark mode", gradientes metálicos e tipografia elegante.
-   **Animações de Scroll**: Uso intensivo de Framer Motion para revelar elementos conforme a navegação.
-   **Performance Otimizada**: Imagens com `loading="lazy"`, carregamento condicional de vídeo (desktop vs mobile) e métricas Core Web Vitals otimizadas.
-   **Comparativo Interativo**: Componente "Antes e Depois" com slider para demonstrar a eficácia dos tratamentos.
-   **SEO Local**: Meta tags e dados estruturados (JSON-LD) configurados para maximizar a visibilidade em buscas locais.
-   **Componentização Modular**: Código organizado em `sections`, `layout` e `ui` para fácil manutenção e escalabilidade.

## Guia de Instalação

Siga os passos abaixo para rodar o projeto localmente.

### Pré-requisitos

-   Node.js versão 18 ou superior.

### Comandos

1.  **Instalar dependências:**
    ```bash
    npm install
    ```

2.  **Iniciar servidor de desenvolvimento:**
    ```bash
    npm run dev
    ```

3.  **Gerar build de produção:**
    ```bash
    npm run build
    ```
